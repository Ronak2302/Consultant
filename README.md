# Ronak Suresh — Marketing Consultant

A clean, modern website for Ronak Suresh, a marketing consultant specializing in clarity-driven brand strategy for founders and startups.

## Features

- Clean, dark-themed design with elegant typography
- Smooth scrolling and smooth reveal animations on scroll
- Interactive sections:
  - From Chaos to Clarity: interactive nodes showing interconnected marketing elements
  - Growth Framework: interactive 5-stage framework (Discover, Diagnose, Position, Activate, Scale)
  - Brands: shows the progression of building a brand from idea to growth
  - About section with portrait and stats
  - Final CTA section
- Fully responsive design
- Built with HTML, CSS, and vanilla JavaScript (no frameworks)

## Sections

1. Hero
2. From Chaos to Clarity (interactive chaos nodes)
3. How I Approach Growth (interactive 5-stage framework)
4. Building Brands (brand progression visualization)
5. About
6. Final CTA
7. Footer

## Technologies

- HTML5
- CSS3 (CSS variables, CSS Grid, Flexbox)
- Vanilla JavaScript (IntersectionObserver for animations, event listeners)
- Google Fonts (Inter and Inter Tight)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ronak-suresh-apple-consultant.git
   ```
2. Open `index.html` in your browser.

## Deployment

This is a static site and can be deployed to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
